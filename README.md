# Resumen del proyecto

**Blockchain básico con Python**: Este proyecto trata sobre como se puede desarrollar un pequeño programa de Blockchain usando Python. 

# Objetivos del Proyecto

Este proyecto, sigue **dos objetivos** de aprendizaje:

1. Diseñar un blockchain con Python. 
2. Entender los conceptos básicos de un blockchain.
 
Al final de este proyecto, podrás comprender cómo funcionan las cadenas de bloques para luego trabajar sobre ellas.

# Requisitos

El único requisito es tener conocimientos básicos de POO en Python.

# Requerimientos

- Python 3
- Libreria: hashlib, time, json

# Estructura del proyecto

El proyecto práctico de **Blockchain básico con Python** se divide en las siguientes tareas:

- Crear el archivo js con su editor de preferencia.
- Ejecutar el codigo, probar y corregir errores.
